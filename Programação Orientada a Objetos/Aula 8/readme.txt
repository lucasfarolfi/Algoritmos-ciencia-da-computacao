--------- Generics ---------

É o substituto do casting, deixa o código mais limpo

como utilizar: 
Ex de casting:

String a = (String) b;

Ex de generics:

Classe<String> b = new Classe()<String>;



------- Classes wrapper ---------

Servem para criar variáveis de tipos primitivos como um objeto

Ex: int a = 1;

ou

Int a = new Int(1);
